How to Add Content ➕

Keep your study hub updated easily — from any device

📁 Method 1 — Add Files via GitHub (Recommended)

1
Go to github.com/RealistsDevLab/DCSYear1Sem2 and open the repository
2
Click Add file → Upload files — drag your PDF, DOCX, or image
3
Write a commit message e.g. Add CSC122 past paper 2025 and click Commit
4
Open index.html in the repo, click the pencil ✏️ to edit
5
Find the course section and add a new resource item with the file path — commit changes

📧 Method 2 — From Your Email (Forward & Upload)

1
Receive notes or papers in your email
2
Download the attachment to your phone or PC
3
Open GitHub on your phone browser → go to the repository → Upload the file
4
Update index.html to add the new resource link

💻 Method 3 — From Terminal (Fastest for You)

1
Clone the repo: git clone https://github.com/RealistsDevLab/DCSYear1Sem2.git
2
Copy your new file into the folder, then: git add .
3
git commit -m "Add new resource"
4
git push — done! Site updates in 1-2 minutes
